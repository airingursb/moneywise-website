# MoneyWise 계정 삭제하기

**앱:** MoneyWise · **개발자:** Airing · **시행일:** 2026-05-01

이 페이지에서는 MoneyWise 계정을 영구적으로 삭제하는 방법과 삭제 후 데이터 처리 방식을 설명합니다.

## 두 가지 삭제 방법

### 방법 1: 앱 내에서 삭제 (권장)

1. **MoneyWise** 앱을 엽니다.
2. **"마이"** 탭을 탭합니다 (또는 **설정** 을 엽니다).
3. 상단 프로필 카드 →  **"내 계정"** 을 탭합니다.
4. **"위험 구역"** 까지 스크롤 →  **"계정 삭제"** 를 탭합니다.
5. 확인 대화상자를 읽고  **DELETE** 를 입력하여 확인합니다.
6. **"계정 삭제"** 를 다시 탭합니다. 앱이 자동으로 로그아웃되고 로그인 화면으로 돌아갑니다.

삭제는 즉시 완료됩니다. 이메일 추가 확인 단계는 없습니다. 앱 내에서 확정하는 순간 계정이 삭제됩니다.

### 방법 2: 이메일 요청

앱에 접근할 수 없는 경우 (예: 앱을 삭제했지만 서버 측 데이터까지 지우고 싶은 경우), MoneyWise 가입에 사용한 이메일 주소에서  **moflowapp@gmail.com**  으로 제목  **"Delete my MoneyWise account"** 로 메일을 보내주세요.

본인 확인 후 14 일 이내에 삭제를 완료합니다.

## 삭제되는 데이터

계정 삭제 시 다음 데이터가 **영구적이고 복구 불가능하게** 제거됩니다:

**서버 (클라우드):**
- MoneyWise 인증 정보 (이메일, Google / Apple 로그인 연결).
- 클라우드 동기화된 모든 레코드: 계좌, 거래, 예산, 투자, 반복 규칙, 카테고리 설정.
- 서버에 저장된 영수증 및 첨부 이미지.
- 아바타 이미지.
- 사용자 설정 (클라우드 동기화 시 저장한 AI 제공업체 API 키 포함).
- 공유 가족 워크스페이스에서의 멤버십.

**기기 내:**
- 로컬 세션이 지워지고 앱이 로그아웃됩니다.
- 암호화된 로컬 SQLite 데이터베이스는 앱을 삭제하거나 기기의 로컬 데이터를 지울 때까지 기기에 남아 있습니다. 이는 사용자의 기기 내 데이터이며 당사에서 접근할 수 없습니다. 로컬 데이터를 즉시 지우려면 휴대폰에서 앱을 삭제하세요.

**제3자 연결:**
- Google / Apple 로그인 연결은 당사 쪽에서 해제됩니다. 제3자가 보유한 토큰은 자연 만료되며, Google 계정 설정 또는 Apple ID 설정에서 수동으로 취소할 수도 있습니다.

## 보관되는 데이터와 보관 기간

- **사용자가 소유한 가족 워크스페이스**: 공유 가족 워크스페이스의 소유자인 경우, 다른 멤버가 필요한 데이터를 내보낼 수 있도록 해당 워크스페이스의 데이터는 **30 일간** 보관됩니다. 30 일 후 가족 워크스페이스 데이터는 완전히 삭제됩니다.
- **익명 분석 데이터**: 삭제 이전에 수집된 집계 값 (페이지 조회, 버튼 탭 횟수)은 분석 파이프라인에 익명 집계 형태로 남아 있습니다. 삭제 완료 후에는 사용자의 신원과 연결되지 않습니다.
- **그 외에는 보관하지 않습니다.** 위 가족 유예 기간을 제외하고, 거래, 잔액, AI 대화 기록의 백업을 일체 보관하지 않습니다.

## 계정 삭제는 활성 구독을 취소하지 않습니다

MoneyWise 계정을 삭제해도 App Store 또는 Google Play 의 활성 구독은 **자동으로 취소되지 않습니다**. 결제를 중단하려면:

- **iPhone / iPad**: 설정 → 본인 이름 →  **"구독"** → MoneyWise →  **"구독 취소"**.
- **Android**: Google Play 스토어 → 프로필 →  **"결제 및 정기 결제"** →  **"정기 결제"** → MoneyWise →  **"해지"**.

결제를 중단하려면 계정을 삭제하기 **전에** 구독을 먼저 취소하세요.
