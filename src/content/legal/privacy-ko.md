# 개인정보 처리방침

**시행일:** 2026-04-18

본 방침은 MoneyWise가 사용자의 데이터를 어떻게 처리하는지 설명합니다.

## 요약 (TL;DR)

- 모든 금융 기록은 암호화된 로컬 데이터베이스로 **사용자의 기기에** 저장됩니다.
- 앱 내에서는 **개인을 식별할 수 있는 분석 데이터를 일절 수집하지 않습니다** — 크래시 리포터도, 트래킹 SDK도 사용하지 않습니다.
- **클라우드 동기화**를 활성화하면 데이터는 기기를 떠나기 전에 암호화됩니다. 암호화된 행은 Supabase에 저장되며, 저희는 그 내용을 읽을 수 없습니다.
- 본인의 API 키로 **AI 어드바이저**를 사용할 경우, 요청은 기기에서 AI 제공자에게 직접 전송됩니다. 저희는 그 요청을 중계하거나 기록하거나 라우팅하지 않습니다.
- 이 웹사이트(`moneywise.airingdeng.com`)는 프라이버시를 존중하는 쿠키 없는 페이지뷰 분석을 위해 **Umami**를 사용합니다. IP 주소나 개인 식별 정보는 저장되지 않습니다.

## MoneyWise가 수집하는 정보

**사용자의 기기 안에서**(동기화를 활성화하지 않는 한 전송되지 않음):
- 계정·잔액·거래 기록·예산·투자 보유 종목.
- 사용자가 첨부하기로 선택한 영수증 이미지.
- AI 제공자의 API 키 (iOS 키체인에 저장).

**클라우드 동기화가 활성화된 경우**(Pro 구독자):
- 위의 데이터를 암호화한 행 (저희가 운영하는 Supabase 서버에 저장). 암호화 키는 기기에 남아 있습니다.
- Supabase 계정용 이메일 주소 (로그인 용도로만 사용).
- 동기화 시 다른 기기를 식별하기 위한 기기 식별자.

**Pro의 공용 AI 크레딧을 사용할 경우**(추후 기능, v1에는 포함되지 않음):
- 사용자의 프롬프트와 어드바이저의 응답은 저희 Supabase Edge Function을 거쳐 라우팅됩니다. 로그는 남용 방지를 위해 7일간 보관된 뒤 삭제됩니다.

**이 웹사이트에서:**
- Umami가 기록하는 항목: 조회한 페이지·리퍼러·국가(대략)·브라우저/OS(대략)·화면 크기. 개인을 식별하는 정보는 포함되지 않습니다.
- 쿠키 없음. 트래킹 픽셀 없음. 광고 네트워크 없음.

## MoneyWise가 수집하지 **않는** 정보

- 정확한 위치 정보를 수집하지 않습니다.
- 데이터를 제3자에게 공유·판매·임대하지 않습니다.
- 광고를 게재하지 않습니다.
- 사용자의 금융 기록을 읽지 않습니다. 클라우드 동기화 데이터는 암호화되어 있어 저희에게는 불투명한 바이트 열로만 보입니다.

## 사용자의 권한

- **내보내기**: 앱 내에서 모든 기록을 CSV로 내보낼 수 있습니다.
- **삭제**: 앱을 삭제하면 모든 로컬 데이터가 제거됩니다. 클라우드 동기화를 사용했다면 아래 연락처로 삭제 요청을 보내주세요.
- **동기화 중지**: 설정에서 클라우드 동기화를 끄면 30일 이내에 서버에서 암호화된 행 데이터가 삭제됩니다.
