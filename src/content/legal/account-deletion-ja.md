# MoneyWise アカウントを削除する

**アプリ：** MoneyWise · **開発者：** Airing · **施行日：** 2026-05-01

このページでは、MoneyWise アカウントを完全に削除する方法と、削除後のデータの取り扱いを説明します。

## 2 つの削除方法

### 方法 1：アプリ内から削除（推奨）

1. **MoneyWise** アプリを開きます。
2. **「マイページ」** タブをタップします（または **設定** を開きます）。
3. 上部のプロフィールカード →  **「マイアカウント」** をタップします。
4. **「危険ゾーン」** までスクロール →  **「アカウントを削除」** をタップします。
5. 確認ダイアログを読み、 **DELETE** と入力して確認します。
6. もう一度 **「アカウントを削除」** をタップします。アプリは自動的にサインアウトしてログイン画面に戻ります。

削除は即座に完了します。メール確認の手順はありません。アプリ内で確定すると、アカウントは削除されます。

### 方法 2：メールでリクエスト

アプリにアクセスできない場合（アンインストール済みだがサーバー側のデータを削除したい場合など）、MoneyWise に登録したメールアドレスから  **moflowapp@gmail.com**  宛に件名  **「Delete my MoneyWise account」**  でメールをお送りください。

本人確認のうえ、14 日以内に削除を完了します。

## 削除されるデータ

アカウント削除後、以下のデータは **永続的かつ復元不可** に削除されます：

**サーバー側（クラウド）：**
- MoneyWise の認証情報（メールアドレス、Google / Apple サインインの連携）。
- クラウド同期されたすべてのレコード：口座、取引、予算、投資、定期ルール、カテゴリ設定。
- サーバーに保存されたレシート画像と添付ファイル。
- アバター画像。
- ユーザー設定（クラウド同期時に保存した AI プロバイダーの API キーを含む）。
- 共有家族ワークスペースのメンバーシップ。

**端末内：**
- ローカルセッションがクリアされ、アプリはサインアウトされます。
- 暗号化されたローカル SQLite データベースは、アプリをアンインストールするか端末のローカルデータを消去するまで端末に残ります。これはあなたの端末内のデータで、当方からアクセスすることはできません。すぐにローカルデータを消去したい場合は、アプリを端末から削除してください。

**サードパーティ連携：**
- Google / Apple サインインによる連携は当方側で解除されます。第三者が保持しているトークンは自然に失効しますが、Google アカウント設定や Apple ID 設定から手動で取り消すこともできます。

## 保持されるデータと保持期間

- **あなたが所有する家族ワークスペース**：共有家族ワークスペースのオーナーである場合、他のメンバーが必要なデータをエクスポートできるよう、そのワークスペースのデータは **30 日間** 保持されます。30 日経過後、家族ワークスペースのデータは完全に削除されます。
- **匿名化された分析データ**：削除前に収集された集計値（ページビュー、ボタンタップ数）は、匿名の集計として分析パイプラインに残ります。削除完了後はあなたの ID と紐づきません。
- **それ以外は保持しません。** 上記の家族の猶予期間を除き、取引、残高、AI 会話履歴のバックアップは一切保持しません。

## アカウント削除は有効なサブスクリプションを解約しません

MoneyWise アカウントを削除しても、App Store または Google Play の有効なサブスクリプションは **自動的に解約されません**。請求を止めるには：

- **iPhone / iPad**：「設定」 → 自分の名前 → **「サブスクリプション」** → MoneyWise → **「サブスクリプションをキャンセル」**。
- **Android**：Google Play ストア → プロフィール → **「お支払いと定期購入」** → **「定期購入」** → MoneyWise → **「定期購入を解約」**。

請求を止めたい場合は、アカウント削除の **前** にサブスクリプションを解約してください。
